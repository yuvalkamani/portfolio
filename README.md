# Portfolio Website

This repository contains the code for my personal portfolio website built using React and CSS. It showcases my projects, skills, and provides an overview of my experience and qualifications.

## Installation

To run the portfolio website locally, follow these steps:

1. Clone this repository to your local machine using:

   ```bash
   git clone https://github.com/yuvalkamani/portfolio.git
   ```

2. Navigate to the project's directory:

   ```bash
   cd portfolio
   ```

3. Install the required dependencies using npm:

   ```bash
   npm install
   ```

## Usage

After completing the installation steps, you can run the website on your local machine:

```bash
npm start
```

This command starts the development server and opens the website in your default browser. Any changes you make to the code will automatically reload the website, allowing you to see the updates in real-time.

To customize the website content, design, and layout, you can modify the relevant files in the project.

## Technologies Used

The portfolio website is built using the following technologies:

- React
- CSS
- HTML

Additional dependencies and libraries can be found in the `package.json` file.
